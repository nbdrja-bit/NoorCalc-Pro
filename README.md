# 🧮 NoorCalc Pro

A modern scientific calculator built using Python and Kivy.

# 🚀 Features

* Keyboard & Button support
* Full expression input (like mobile calculators)
* Scientific functions (√, x², %, power)
* Calculation history
* Clean UI design


# 🛠️ Technologies Used

* Python
* Kivy

# ▶️ How to Run

pip install kivy
python main.py


# 🎯 Future Improvements

* Dark/Light mode
* More scientific functions (sin, cos, log)
