# NoorCalc-Pro
 🧮 NoorCalc Pro  A modern scientific calculator built using Python and Kivy.  ## 🚀 Features  * Keyboard &amp; Button support * Full expression input (like mobile calculators) * Scientific functions (√, x², %, power) * Calculation history  ##  ▶️ How to Run  ```bash pip install kivy python main.py ```   ## 👨‍💻 Author  Noor Badr
